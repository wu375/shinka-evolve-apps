This repo contains examples of using ShinkaEvolve for various tasks via Agent Skills.

Currently examples/experiments include:

- `create_figure`: evolve research figures/diagrams, guided by image-only LLM judges.
- `repo_scan`: scan research and ML codebases for snippets worth evolving.
- `self_play_game`: improve game agents against moving opponent pools of baseline and past opponents. Specifically plays a Kaggle game called Orbit Wars.
- `self_play_novelty`: evolve visually novel images. Instead of scoring a candidate image in isolation, judge in a self-play style.
